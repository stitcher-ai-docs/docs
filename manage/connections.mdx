A [Data Connection](/general/glossary) is a configured link to a data source or a destination. Sources bring cost and business data into a [pipeline](/general/glossary); destinations are where outputs are delivered.

A connector describes how data is accessed. For example, a dataset stored in a S3 bucket is read through an S3 connector.
