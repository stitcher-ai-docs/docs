The first time you open a [pipeline](/general/glossary) for editing, you set up the [authoring environment](/general/glossary) in Visual Studio Code. This is a one-time setup per machine.

<Steps>
  <Step title="Open the pipeline for editing">
    From the **Pipelines** page, click the GitHub icon on a pipeline and select **Edit pipeline configuration**. This opens the pipeline's repository in Visual Studio Code.
  </Step>
  <Step title="Install the StitcherAI Editor">
    Install the **StitcherAI Editor** extension (published by StitcherAI Inc.), the visual editing plugin for Visual Studio Code. You can find it in the Extensions panel.
  </Step>
  <Step title="Log in">
    Once the extension loads, the **Welcome to StitcherAI** view opens. Select **Log In** to authenticate.
  </Step>
</Steps>

Once setup is complete, you're ready to build. From the Welcome view you can **Start a Chat**, open the **Quick Start Guide**, or **Manage Pipeline**. See the [Walkthrough](/get-started/walkthrough) for a guided first task, or [Access your repository](/building/access-your-repository) for the authoring methods.
